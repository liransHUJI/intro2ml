Introduction to Machine Learning (67577) - HUJI
This repository contains the course materials, theoretical exercises, and programming projects for the Introduction to Machine Learning course at the Hebrew University of Jerusalem. The course provides a rigorous foundation in the mathematical and statistical principles of machine learning.

Course Overview
The course covers the lifecycle of learning algorithms, from theoretical guarantees (PAC Learning) to practical implementation using Python.

Key Topics
Foundations: PAC Learning, VC Dimension, ERM, and SRM.

Supervised Learning: Linear Regression, Ridge/Lasso, LDA, QDA, SVMs, and Decision Trees.

Ensemble Methods: Boosting (AdaBoost) and Bagging.

Unsupervised Learning: K-Means, PCA, and Hierarchical Clustering.

Deep Learning: Neural Networks and Backpropagation basics.
